The **`pi()`** function returns the value of PI.

## Syntax

```pseudo
pi();
```

The `pi()` function has no parameters. It returns the value of PI as float

## Example

```php
<?php
  echo pi();
?>
```

This will output:

```shell
// 3.1415926535898
```
