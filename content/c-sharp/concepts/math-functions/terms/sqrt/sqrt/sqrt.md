**Math.sqrt()** returns the square root of any number set in the parameter

Note: The number must be greater than or equal to 0.

## Syntax
math.sqrt(x)

Parameter: (x) is the number to find the square root of.If the number is less than 0, it returns a Value Error.If the value is not a number,it returns a TypeError

## Example
# Import math
# Return the square root of different numbers

## Codebyte Example
print(math.sqrt(16)) outputs 4.0
