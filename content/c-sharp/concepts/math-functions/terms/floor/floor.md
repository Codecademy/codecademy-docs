**Math.floor()** is a static method that always rounds down and returns the largest whole integer less than or equal to the given number.

## Syntax
```pseudo
Math.floor(x);
```


## Example

```cs
public class Floor {
    public static void example() {
        Math.floor(5.05);
        //returns 5
    }
}
```
