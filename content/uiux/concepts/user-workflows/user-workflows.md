User workflows are navigational paths throughout a digital interface. User workflows illustrate the path a user takes to complete a task, and there may be multiple workflows that different users take to complete the same task.

![Information Architecture - Site Map example](https://static-assets.codecademy.com/Courses/intro-to-ui-and-ux/wireframes/information-architecture.svg)

This image features a [site map](https://www.codecademy.com/resources/docs/uiux/site-map). In this site map, a path from homepage -> resources -> projects is considered a user workflow.
