## Definition
***
The `randint()` method 

## Syntax
***
random.randint()

## Example 1
***
Use `randint()` to return a random number between 0 and 99:
```codebyte/python
import random

print(random.randrange(0, 100))
```

## Example 2
***
Use `randrange()` to return a random number between 0 and 99 and is divisible by 5:
```codebyte/python
import random

print(random.randrange(0, 100, 5))
```
