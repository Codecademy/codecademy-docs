## Definition
***
Returns the `float` value of `x` raised to the power of `y`. This contrasts with the built-in `**` operator in that `math.pow()` converts both its arguments to type `float`.

## Syntax
***
```py
math.pow(x, y)
```

## Example 1
***
Use `math.pow()` to return `5` to the power of `3`:
```codebyte/python
# Import math library
import math

print(math.pow(5, 3))
```

## Example 2
***
Use `math.pow()` to return `5.5` to the power of `3.3`:
```codebyte/python
# Import math library
import math

print(math.pow(5.5, 3.3))
```
