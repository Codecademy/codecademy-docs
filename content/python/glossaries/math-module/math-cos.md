## Definition

Returns the cosine of `x` radians.

## Syntax

```py
math.cos(n)
```

## Example 1

Use `math.cos()` to return the cosine of a `90` degree angle:
```codebyte/python
import math

print(math.cos(90))
```

## Example 2

Use `math.cos()` to return the cosine of a `180.5` degree angle:

```codebyte/python
import math

print(math.cos(180.5))
```
