## Definition
***
The `math.cos()` function returns the cosine of `x` radians.
## Syntax
***
```py
math.cos(n)
```

## Example 1
***
Use `math.cos()` to return the cosine of a `134.2` degree angle:
```codebyte/python
# Import math library
import math

print(math.cos(134.2))
```

## Example 2
***
Use `math.cos()` to return the cosine of a `412.959` degree angle:
```codebyte/python
# Import math library
import math

print(math.cos(412.959))
```
