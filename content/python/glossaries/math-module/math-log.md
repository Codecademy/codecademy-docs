## Definition
Returns the natural logarithm of `x` (to base e).

## Syntax
```py
math.log(x)
```

## Example 1
Use `math.log()` to return the natural log of `12`(to base e):
```codebyte/python
import math


print(math.log(12))
```

## Example 2
Use `math.log()` to return the natural log of `12`, given base `5`(log(x)/log(base)):
```codebyte/python
import math


print(math.log(5, 12))
```
