## Definition
***
The `math.exp()` function returns E(approximately 2.718282) raised to the power of x.
## Syntax
***
```py
math.exp(x)
```

## Example 1
***
Use `math.exp()` to return E raised to the power of `5`:
```codebyte/python
# Import math library
import math

print(math.exp(5))
```

## Example 2
***
Use `math.exp()` to return E raised to the power of `3`:
```codebyte/python
# Import math library
import math

print(math.exp(3))
```
