## Definition
***
The built-in `float()` takes in a numeric, string, or no value and a copy of that value in the floating point data type.

## Syntax
***
```py
float()
```

## Example 1
***
Use `float()` to change the datatype of an integer value `314` to float:
```codebyte/python
print(float(314))
```

## Example 2
***
Use `float()` to change the datatype of a string value `"314"` to float:
```codebyte/python
print(float("314"))
```

## Example 3
***
Use `float()` to create a new float value:
```codebyte/python
f = float()
print(f)
```
