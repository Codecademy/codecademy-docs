## Definition

Converts a value to a Boolean `True` or `False` value.

Statements are `True` if:

## Syntax

```py
bool(object)
```

## Example 1

Use `bool()` to return if the string 'True' is `True` or `False`:

```codebyte/python
print(bool('True'))
```

## Example 2

Use `bool()` to :

```codebyte/python
print(bool('False'))
```

## Example 3

Use `bool()` to :

```codebyte/python
print(bool(False))
```
