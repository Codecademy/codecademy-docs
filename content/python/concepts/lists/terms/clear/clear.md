## Definition 

Removes all items from the list.

## Syntax

```py
list.clear()
```

The `.clear()` method has no parameters.

## Example

To remove all items from the `orders` list:

```codebyte/python
orders = ['daisies', 'periwinkle']

orders.clear()

print(orders)
# Output: []
```
