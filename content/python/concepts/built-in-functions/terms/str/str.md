Takes in a value that can be converted into a string, and returns a copy of the value in the `string` datatype.

## Syntax

```py
str()
```

## Example 1

Use `str()` to change the datatype of a `float` value `314.0` into a string:

```codebyte/python
print(str(314.0))
```

## Example 2

Use `str()` to change the datatype of an `int` value `314` into a string:

```codebyte/python
print(int(314))
```
