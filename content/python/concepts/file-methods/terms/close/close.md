## Definition

Allows the user to close an open file within the IDE.

## Syntax

```py
file.close()
```

## Example 1

Use `.close()` to close the opened **gullivers_travels.txt** file:

```codebyte/python
f = open("gullivers_travels.txt", "r", encoding='utf8')

f.close()
```
