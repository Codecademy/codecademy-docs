The `.values()` method of a Python dictionary returns list of its values. It takes no arguments.

## Syntax

```pseudo
dictionary.values()
```

## Codebyte Example

The following example creates a dictionary then prints a list of its values.

```codebyte/python
d = {1:'one',2:'two',3:'three'}
d_values = d.values()
print(d_values)
```
