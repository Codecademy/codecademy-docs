The `.values()` method returns all of its values of a Python dictionary. It takes no arguments.

## Syntax

```pseudo
dictionary.values()
```

## Codebyte Example

The following example creates a dictionary, then prints all the values:

```codebyte/python
d = {1:'one',2:'two',3:'three'}
d_values = d.values()
print(d_values)
```
