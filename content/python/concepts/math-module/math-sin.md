## Definition

Returns the sine of input x, in radian value.

## Syntax

```py
math.sin(x)
```

## Example 1

Use `math.sin()` to return the sin of the integer `5` in radian value:

```python
import math

print(math.sin(5))

# Output: -0.9589242746631385
```

## Example 2

Use `math.sin()` to return the sin of the float `5.5` in radian value:

```python
import math

print(math.sin(5.5))

# Output: -0.7055403255703919
```
