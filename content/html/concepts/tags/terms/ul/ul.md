The **`<ul>`** tag represents an unordred list of items.

## Syntax

```pseudo
<ul>
  <li><!-- Content here --></li>
</ul>
```

By default, each list item is prepended with a dot marker. But this can be altered by changing the value of the `list-style-type` property.
