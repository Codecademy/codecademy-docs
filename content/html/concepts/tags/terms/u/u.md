The `<u>` tag is used to mark HTML text to be displayed with an underline or other non-textual annotation. The default rendering of this tag is a simple solid underline, but may be altered using CSS.

## Syntax

```html
<u><!-- text which should be underlined goes here --></u>
```

The `<u>` tag should only contain text content or other tags meant for text content.

## Example

```html
<p>
  <u>Except for this</u>, most of this paragraph is not underlined.
</p>
```
