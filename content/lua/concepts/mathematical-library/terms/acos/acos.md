The Lua.math method which returns the **arc cosine** of the argument, in radians.

## Syntax
```pseudo
math.acos(x)
```

## Example
```lua
math.acos(0.5)
```

A full list of math functions can be found on the [Lua documentation page](https://www.lua.org/manual/5.1/manual.html#5.6).
