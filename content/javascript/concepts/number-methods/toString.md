## Definition

Convert a number to a string.

## Syntax

```js
num.toString();
```

## Examples

Converting number to a string:

```js
const x = 5;

console.log(x.toString());
// Output: 5
```

To verify if it is a string:

```js
const x = 5;

console.log(typeof x.toString());
// Output: string
```
