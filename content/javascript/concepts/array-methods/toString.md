## Definition

Returns a string representing the specified array and its elements. This does not modify original array.

## Syntax

```js
array.toString();
```

The `toString()` method does not have any parameters.

## Examples

```js
const numbers = [1, 2, 3];

console.log(numbers.toString());
// Output: '1,2,3'

console.log(numbers);
// Output: [1, 2, 3]
```
