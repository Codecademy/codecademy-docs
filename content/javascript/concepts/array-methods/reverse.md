## Definition

Reverses the order of the elements of an array in place and returns the reversed array.

## Syntax

The `reverse()` method does not take any parameters.

```js
array.reverse();
```

## Examples

Reverse the elements in an array:

```js
const numbers = [5, 2, 9];
numbers.reverse();

console.log(numbers);
// Output: [9, 2, 5]
```
