# What is an Argument?

An argument is the actual value of a variable (aka. the parameter) passed into a function.


## Example

```js
// The parameter is x since it is the variable
function tripleThis(x) {
  return x * 3 
}

// The argument is 6 since it is the actual value
tripleThis(6)
```
