To set the text alignment of inline contents.

## Syntax

```css
text-align: <text-align-value>;
```

The `text-align-value` can be the following:

- `left`
- `center`
- `right`

## Example 1

In the example below, the `p` tag will be set to `center`:

```css
p {
  text-align: center;
}
```

This causes the `p` tag to be positioned centrally on the page:
