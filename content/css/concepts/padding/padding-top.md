## Definition

Defines the width of the top padding for an element.

## Syntax

```css
padding-top: <value>;
```

where `<value>` can be one of the following:

- Length: `20px`
- Percentage: `5%`

## Example 1

Set the top padding of the `h1` element to be `20px`:

```css
h1 {
  padding-top: 20px;
}
```

## Example 2

Set the top padding of the `h1` element to be `5%` relative to the width of the containing block:

```css
h1 {
  padding-top: 5%;
}
```
