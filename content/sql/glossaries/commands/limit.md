## Definition

Specifies, or limits, the maximum number of rows the result set will have.

## Syntax

```sql
SELECT column_name(s)
FROM table_name
LIMIT 20;
```

## Example

The given query will limit the result set to 5 rows from the `movies` table.

```sql
SELECT *
FROM movies
LIMIT 5;
```
