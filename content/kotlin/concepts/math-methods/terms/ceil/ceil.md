**Title:** `.ceil()`

**Description:** "The `.ceil()` method in Kotlin is used to round a floating-point number up to the nearest integer."

**Syntax:** 
  ```kotlin
  kotlin.math.ceil(x: Double)
```

**Example:** 
```kotlin
val x = 3.14
val y = 3.99

println(ceil(x)) // Output: 4.0
println(ceil(y)) // Output: 4.0

```
