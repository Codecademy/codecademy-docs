## Definition

Returns the inverse sine of the argument in radians.

## Syntax

```java
Math.asin(n)
```

## Example 

Use `Math.asin()` to return the inverse sine of `1.0`:

```java
public class Main {
  public static void main(String[] args) {
  
    double x = 1.0;
    
    System.out.println(Math.asin(x));
    // Output: 1.5707963267948966

  }
}
```
