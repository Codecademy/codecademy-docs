JavaBeans are a [class definitions](https://www.codecademy.com/resources/docs/java/classes) following a set of paticular conventions. They are not tied to any Java famework so any Java program can use them. All JavaBeans can be considered [POJOs](https://www.codecademy.com/resources/docs/java/pojo) (Plain old Java objects).
