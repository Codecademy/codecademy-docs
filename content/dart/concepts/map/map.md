## Syntax

The syntax to declare a `map` is -

```pseudo
var identifier = new Map()
```

## Example

Here is an example on how to use `map` in Dart:

```dart
void main() { 
   var details = {'Cat':'jeff','Color':'brown'}; 
   print(details); 
}
```

The above code gives the following output:


```shell
{Cat: jeff, Color: brown}
```
