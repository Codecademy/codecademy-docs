## Syntax

The syntax to declare a map is -

'''
var identifier = new Map()
'''

## Example

Here is an example -

'''dart
void main() { 
   var details = {'Cat':'jeff','Color':'brown'}; 
   print(details); 
}
'''

...producing the following output -


'''
{Cat: jeff, Color: brown}
'''
